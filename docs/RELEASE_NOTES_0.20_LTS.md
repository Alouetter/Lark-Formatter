# Lark-Formatter 0.20 LTS 发布说明

发布日期：2026-03-19

## 一、版本定位

`0.20 LTS` 是一个**毕业论文格式专项修改工具版本**。

本版本的目标不是覆盖所有文档场景，而是集中解决毕业论文/学位论文排版修订中的高频问题，包括：
- 页面与页边距统一
- 标题识别与编号整理
- 目录、题注、表格、页眉页脚规范化
- 输出对比稿与审计报告

> 说明：其余场景的通用化能力、扩展模板体系与更完整的多场景发布，计划在 `1.00` 版本完成。

## 二、适用范围

### 适用
- 毕业论文
- 学位论文
- 与毕业论文结构接近的 `docx` 学术文稿

### 当前不建议使用的场景
- 公文/行政文书
- 通用报告排版
- 多行业通用模板排版
- 非论文导向的复杂混排文档

## 三、本版本可用能力

- 毕业论文模板加载、保存与克隆
- 页面设置与样式统一
- 标题检测、多级编号与目录处理
- 图表题注与表格规范化
- 分节、页眉页脚与页码处理
- Markdown 清理、空白规范、引用关联等实验室能力
- 输出：
  - 排版结果文档
  - 对比稿
  - JSON 报告
  - Markdown 报告

## 四、本次版本收口内容

- 修复 scene prompt 回归校验误报，确保验证逻辑与运行时一致
- 统一版本标识为 `0.20 LTS`
- 发布脚本切换到 `Lark-Formatter_v0.20_LTS.spec`
- 新增发布结构说明与“不上传内容”说明
- 明确产品定位：当前版本仅服务于毕业论文格式修订场景

## 五、验证结果

- 自动化测试：`539 passed`
- 打包验证：通过
- 发布产物目录：`dist/Lark-Formatter_v0.20_LTS/`

## 六、已知限制

- 当前版本的产品说明、模板组织与默认能力均以毕业论文场景为中心
- 不承诺对其他文档类型提供稳定兼容性
- 某些实验室功能仍建议先进行小样本验证后再批量使用
