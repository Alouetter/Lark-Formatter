# fixtures_public

这个目录用于存放 **可以公开再分发** 的最小化测试素材。

## 允许放入的内容
- 代码动态生成的 `.docx` / `.md` / `.json` 最小样本
- 你本人原创并明确允许公开的 synthetic fixtures
- 已脱敏、无法回溯到真实个人 / 机构的最小复现样本

## 禁止放入的内容
- 高校、期刊、机构规范原文或其大段摘录
- 真实论文、真实报告、真实业务文档
- 带真实姓名、学号、院系、单位、项目名称的信息
- 运行产物，如 `*_new.docx`、`*_对比稿.docx`、报告和日志

## 建议
- 每个 fixture 尽量只覆盖一个问题。
- 如能用代码生成，就不要上传二进制样本。
- 若来源或版权不明确，请默认不要加入公开仓库。
